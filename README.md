# React Photo Gallery
A responsive photo gallery application built using React, Vite, and Tailwind CSS.
The application fetches images from the Picsum Photos API, displays them in a responsive grid layout, allows users to search photos by author name, and enables users to mark photos as favourites.
Favourite photos are stored in localStorage, ensuring they persist even after the page is refreshed.

---

## Features

- Fetch photos from an external API
- Loading and error state handling
- Responsive grid layout for different screen sizes
- Real-time search filtering by author name
- Toggle favourite photos using a heart icon
- Favourite photos persist using localStorage
- Custom React hook for API data fetching
- State management using useReducer
- Performance optimization using useMemo and useCallback

## Installation & Setup

- Clone the repository:
- git clone https://github.com/your-username/photo-gallery.git
- Navigate to the project directory:
- cd photo-gallery
- Install dependencies:
- npm install
- Start the development server:
- npm run dev
